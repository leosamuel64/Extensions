# Groupement d'extension

## Installation

```Bash
>> cd Extensions
>> sudo ./install
```


## jupconvert
Utilitaire de convertion entre jupyter Notebook vers PDF.  

Utilisation :  
```Bash
>> jupconvert FICHIERORIGINE NOMDUFICHIER
```

## latexbuilder
Compilateur LATEX

Utilisation :  
```Bash
>> latexbuilder FICHIERORIGINE
```
